# 智能体形象管理功能 Demo

## 项目概述

这是一个智能体动态形象管理功能的HTML演示项目，基于产品经理提供的设计方案开发。该demo完整展示了智能体形象管理的核心业务流程和用户交互体验。

## 功能特性

### 🎯 核心功能

1. **智能体列表管理**
   - 展示智能体基本信息（名称、分类、创建时间等）
   - 支持分页浏览和搜索功能
   - 提供编辑、使用设置、形象管理等操作

2. **动态形象管理**
   - 静态形象展示（同步COZE平台封面）
   - 动态形象开关控制
   - 资源上传管理（背景图 + 4个表情动图）
   - 实时预览和状态提示

3. **玩具端预览**
   - 模拟真实玩具设备屏幕
   - 支持静态/动态形象切换
   - 表情状态实时控制
   - 自动演示功能

### 🔧 技术特性

- **纯前端实现**：使用HTML5 + CSS3 + JavaScript
- **响应式设计**：适配桌面和移动设备
- **模块化架构**：清晰的代码组织和状态管理
- **交互友好**：丰富的动画效果和用户反馈

## 业务流程

### 1. 基础关联流程
- 智能体创建时同步COZE平台封面作为静态形象
- 后端维护智能体ID与形象配置的关联关系
- 玩具端根据配置决定加载静态或动态形象

### 2. 动态形象配置流程
- 默认关闭动态形象，展示静态形象
- 用户开启动态形象开关
- 上传背景图和4个表情动图（聆听、回答、待机、空闲）
- 系统校验资源完整性
- 保存配置并启用动态形象

### 3. 资源加载与降级
- 动态形象开启且资源完整：优先加载动态形象
- 动态形象关闭或资源不完整：自动降级为静态形象
- 前端管理页面提供清晰的状态提示

## 使用指南

### 快速开始

1. **启动项目**
   ```bash
   # 直接在浏览器中打开 index.html
   # 或使用本地服务器
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

2. **浏览智能体列表**
   - 查看现有智能体信息
   - 使用分页功能浏览更多内容
   - 点击"形象管理"按钮进入配置页面

### 形象管理操作

1. **静态形象管理**
   - 查看当前静态形象（同步COZE平台）
   - 使用AI生成功能（演示）

2. **动态形象配置**
   - 开启"启用动态形象"开关
   - 上传背景图（建议800x600px，JPG/PNG格式）
   - 上传4个表情动图：
     - 聆听表情（GIF格式，建议200x200px）
     - 回答表情（GIF格式，建议200x200px）
     - 待机表情（GIF格式，建议200x200px）
     - 空闲表情（GIF格式，建议200x200px）
   - 点击"保存配置"完成设置

3. **预览效果**
   - 点击"预览效果"按钮进入玩具端模拟
   - 切换显示模式（静态/动态）
   - 手动控制表情状态
   - 使用自动演示功能

### 玩具端预览

1. **显示模式切换**
   - 静态形象：展示智能体头像
   - 动态形象：背景图叠加表情动图

2. **表情控制**
   - 聆听：智能体处于听取用户输入状态
   - 回答：智能体正在回应用户
   - 待机：智能体准备就绪状态
   - 空闲：智能体无活动状态

3. **自动演示**
   - 开启后每3秒自动切换表情
   - 模拟真实使用场景
