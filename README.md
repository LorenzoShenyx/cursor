# 智能体动态形象管理系统 Demo

这是一个基于您提供的Axure原型设计方案开发的智能体动态形象管理功能演示系统，完全使用HTML、CSS和JavaScript实现，可直接在浏览器中预览和体验。

## 🚀 功能特性

### 1. 智能体列表管理
- **智能体概览**：展示所有智能体的基本信息和状态
- **分类管理**：支持教育辅导、娱乐互动、生活助理、健康医疗等分类
- **状态监控**：实时显示智能体运行状态和形象配置状态
- **统计面板**：直观显示活跃智能体数量、配置状态等关键指标

### 2. 动态形象配置
- **背景图上传**：支持JPG、PNG格式，建议尺寸800x600
- **表情动图管理**：
  - 聆听表情：智能体聆听用户输入时的动图
  - 回答表情：智能体回答问题时的动图
  - 待机表情：智能体空闲状态的动图
  - 加载静态图：系统加载时的占位图片
- **实时预览**：上传后立即在右侧预览面板显示效果
- **状态切换**：可切换不同状态查看对应表情效果

### 3. 玩具端模拟预览
- **真实屏幕模拟**：320x480分辨率，模拟真实玩具屏幕
- **状态指示器**：显示当前智能体状态（聆听/回答/待机）
- **动画效果**：包含波浪动画、状态切换动画等
- **全屏预览**：支持全屏模式，更好地展示效果
- **自动演示**：无交互时自动循环切换状态

### 4. 交互体验优化
- **文件上传验证**：文件大小、格式检查
- **保存同步模拟**：模拟真实的保存和同步过程
- **错误处理**：友好的错误提示和异常处理
- **响应式设计**：适配不同屏幕尺寸

## 🎯 使用指南

### 1. 启动系统
直接用浏览器打开 `index.html` 文件即可开始使用。

### 2. 基本操作流程

#### 步骤1：选择智能体
1. 在智能体列表页面查看所有可用的智能体
2. 点击任意智能体行的"形象管理"按钮

#### 步骤2：配置动态形象
1. **上传背景图**：点击背景图上传区域，选择合适的背景图片
2. **配置表情动图**：
   - 点击各个表情上传区域
   - 分别上传聆听、回答、待机、加载等状态的图片/动图
3. **实时预览**：右侧预览面板会实时显示上传效果
4. **状态切换**：点击"聆听"、"回答"、"待机"按钮查看不同状态效果

#### 步骤3：保存和预览
1. 点击"保存并同步到玩具端"保存配置
2. 点击"打开玩具端预览"在新窗口查看最终效果
3. 在玩具端预览页面可以：
   - 切换不同状态查看效果
   - 使用全屏模式
   - 模拟加载过程

### 3. 高级功能

#### 玩具端预览模式
- **状态切换**：使用底部控制面板切换智能体状态
- **全屏预览**：点击右上角全屏按钮获得沉浸式体验
- **自动演示**：系统会自动循环展示不同状态（10秒间隔）
- **模拟加载**：点击"模拟加载"体验真实的加载过程

#### 文件格式要求
- **背景图**：JPG、PNG格式，建议800x600像素
- **表情动图**：GIF格式优先，建议200x200像素
- **文件大小**：单个文件不超过5MB
