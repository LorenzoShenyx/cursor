# 智能体动态形象管理功能 Demo

这是一个基于HTML/CSS/JavaScript的智能体动态形象管理功能演示项目，完全按照您提供的Axure原型设计方案实现。

## 功能概述

本demo实现了智能体动态形象管理的完整功能流程：

1. **智能体列表页** - 展示智能体列表，新增"形象管理"按钮
2. **形象管理页** - 动态面板实现多状态切换，支持静态/动态形象配置
3. **玩具端预览页** - 模拟玩具屏幕，展示静态/动态形象效果

## 使用方法

### 1. 启动Demo
直接在浏览器中打开 `index.html` 文件即可开始体验。

### 2. 功能演示流程

#### 步骤1：查看智能体列表
- 打开 `index.html`
- 查看智能体列表，每个智能体都有"形象管理"按钮

#### 步骤2：进入形象管理
- 点击任意智能体的"形象管理"按钮
- 跳转到形象管理页面

#### 步骤3：体验不同状态
**智能体1001（静态模式）**：
- 默认显示静态形象
- 点击"开启动态形象"进入配置状态

**智能体1002（动态关闭）**：
- 显示动态形象已关闭状态
- 可以开启开关进入配置

**智能体1003（动态完整）**：
- 直接显示动态形象配置状态
- 所有资源已上传完成

#### 步骤4：配置动态形象
- 上传背景图（点击或拖拽）
- 上传4个表情动图
- 实时查看上传状态
- 资源完整后保存配置

#### 步骤5：预览效果
- 点击"玩具端预览"按钮
- 在新窗口查看玩具端效果
- 测试不同状态切换

### 3. 交互说明

#### 文件上传
- **支持格式**：PNG、JPEG、JPG
- **文件大小**：不超过5MB
- **上传方式**：点击上传区域或拖拽文件

#### 状态切换
- **静态→动态**：点击"开启动态形象"按钮
- **动态→静态**：点击"关闭动态形象"按钮
- **开关控制**：使用开关组件切换状态

#### 预览功能
- **静态模式**：显示静态形象
- **动态模式**：背景图+表情动图叠加
- **状态切换**：点击控制按钮切换表情

## 技术实现

### 前端技术栈
- **HTML5**：语义化标签，表单验证
- **CSS3**：Flexbox布局，Grid布局，动画效果
- **JavaScript ES6+**：模块化编程，事件处理

### 核心特性
- **响应式设计**：适配桌面端和移动端
- **动态面板**：纯CSS+JS实现状态切换
- **文件处理**：FileReader API处理图片上传
- **数据管理**：SessionStorage存储配置信息
- **动画效果**：CSS3动画和过渡效果
