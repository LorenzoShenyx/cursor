# 智能体动态形象管理功能 Demo

## 📋 功能概述

这是一个完整的智能体动态形象管理功能HTML演示页面，完全按照产品设计方案实现，包含静态必填逻辑、动态叠加效果和资源校验功能。

## ✨ 核心功能

### 🎯 静态形象管理（必填）
- **COZE平台同步**：自动同步COZE平台封面作为静态形象兜底
- **重新同步功能**：支持手动重新同步最新封面
- **同步状态反馈**：实时显示同步进度和结果

### 🎨 动态形象配置（可选增强）
- **背景图上传**：支持JPG、PNG格式，建议尺寸320x240
- **表情动图管理**：
  - 聆听表情（核心状态）
  - 回答表情
  - 待机表情（核心状态）
- **格式验证**：自动验证文件格式和完整性

### 🔧 叠加效果配置
- **可视化叠加预览**：实时预览背景图与表情动图叠加效果
- **拖拽调整位置**：支持鼠标拖拽调整表情动图位置
- **自动调整功能**：一键调整到最佳安全位置
- **冲突检测**：自动检测叠加冲突并提示

### 📱 玩具端预览
- **实时状态切换**：支持待机/聆听/回答状态预览
- **智能加载逻辑**：
  - 优先显示动态形象（背景+表情动图）
  - 资源不完整时自动降级到静态形象
- **状态指示器**：清晰显示当前加载状态和缺失资源

### ✅ 资源校验系统
- **完整性验证**：检查动态资源配置完整性
- **核心状态提醒**：建议上传核心状态表情（聆听+待机）
- **实时反馈**：上传过程中实时显示校验结果

## 🚀 使用方法

1. **打开页面**：直接在浏览器中打开`agent-avatar-management.html`
2. **点击"新增智能体"**：打开管理弹窗
3. **切换到"形象管理"Tab**：体验完整功能
4. **上传资源**：
   - 先查看静态形象（COZE封面）
   - 上传背景图片
   - 上传表情动图（建议至少上传聆听和待机）
5. **调整叠加效果**：在叠加预览区拖拽调整位置
6. **预览效果**：在玩具端预览区查看最终效果
7. **保存同步**：保存配置并同步到玩具端

## 🎛️ 交互特性

- **Tab切换**：基本信息/形象管理/高级设置
- **文件拖拽上传**：点击上传按钮选择文件
- **实时预览更新**：上传后立即更新预览
- **拖拽调整**：鼠标拖拽调整表情动图位置
- **状态切换**：点击按钮切换不同表情状态
- **智能提示**：根据资源完整性显示相应提示

1. **静态形象兜底**：确保每个智能体都有基础形象

3. **智能加载策略**：优先动态形象，降级静态形象
4. **资源完整性校验**：确保动态形象配置的有效性
